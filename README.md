# Машина Колмогорова–Успенского vs Машина Тьюринга

### Демонстрация предиката Григорьева в реальном времени

Проект реализует сравнение двух моделей вычислений на классической задаче — вычислении **скользящего XOR** по последним `2^L` битам входной последовательности (предикат Григорьева).

## Что демонстрирует проект

-   Машина Тьюринга: простая лента, O(N) шагов на бит после заполнения окна → при большом L выходит из real-time.
-   Машина Колмогорова–Успенского: графовая память, экспоненциальный препроцессинг → зато O(1) операций на бит навсегда.

## Структура проекта

```
.
├── memory.py          # Графовая модель памяти для KUM
├── KUM.py             # Реализация машины Колмогорова–Успенского
├── compare.py         # Интерактивное сравнение + автоматический бэнчмарк с графиками
├── requirements.txt   # Зависимости
└── README.md          # Описание
```

## Как запустить

1.  Склонируйте репозиторий

```bash
git clone https://github.com/Alexander-Snigirev/AltExamMlita
```

2.  Установите зависимости:

```bash
pip install -r requirements.txt
```

3.  Запуск программы

-   Можно запустить отдельно программу для Машины Колмагорова-Успенского для этого:

```bash
python interface.py
```

-   Можно запустить отдельно программу для Машины Тьюринга:

```bash
python MT.py
```

-   Можно запустить сразу 2 программы, чтобы сравнить работу в интерактивном режиме:

```bash
python compare.py
```

-   Можно запустить сразу 2 прогаммы с графиками:

```bash
python compare.py --benchmark
```
